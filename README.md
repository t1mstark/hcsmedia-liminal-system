# HCSMEDIA LIMINAL SYSTEM

Webbasierte Mystery- und Analyseplattform im Stil eines verlassenen Forschungsnetzwerks.

**HCSMEDIA — made with ❤️ by HCSMEDIA**

## Features (MVP)

- Dreisprachig (EN/DE/FR) mit Language-Picker beim ersten Start
- Boot-Interface mit Initialisierungslogs
- Hauptinterface mit Topbar, Navigation, Content- und Data-Panel
- Liminale Weltknoten (Orte, Signale, Anomalien, Koordinaten)
- Atmosphärisches UI (dunkel, scanline/glitch-inspiriert)
- Signatur in Footer und Startbildschirm integriert

## Stack

- React + TypeScript + Vite

## Lokal starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
